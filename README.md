# TICS JetBrains Plugin

This plugin is written for JetBrains IntelliJ based IDE's.

## Features

- Run analysis on:
    - Files
    - Folders / Packages
    - Projects
- Install TICS
- Configure TICS

## Usage

In the tool-menu look for the dropdown `TICS`, there are the options.

_In the old UI buttons are present on the right top._
